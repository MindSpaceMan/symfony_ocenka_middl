### Требования
	•	PHP 8.2
	•	Symfony 7.4
	•	MySQL
	•	Composer

### Установка и запуск
```
composer install
    Настрой подключение к MySQL в .env (DATABASE_URL под свою локальную БД).
    Создай БД и примени миграции:
symfony console doctrine:database:create
symfony console doctrine:migrations:migrate
    Загрузить справочник услуг (1 раз):
php bin/console doctrine:fixtures:load --append
    Запуск приложения:
symfony serve
```
